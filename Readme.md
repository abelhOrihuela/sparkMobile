  ## How to add video


  ### Prerequisites
  import Video from 'react-native-video'
  import lights from '../lights.mp4'


  ```
  <Video
    source={lights}
    resizeMode='cover'
    style={StyleSheet.absoluteFill} />
  <Video
    source={{uri: ''}}
    resizeMode='cover'
    style={StyleSheet.absoluteFill} />
  ```
